---

## INSTALLATION & SETUP

1. Clone the repo:
   git clone https://github.com/your-username/backroad-application.git

2. Navigate into the directory:
   cd backroad-application

3. Install dependencies:
   npm install

4. Run the development server:
   npm start

5. Build the production version:
   npm run build

---
